## 모두연 풀잎스쿨 19기 - 딥러닝 컴퓨터비전 완벽 가이드 소개
* Object Detection과 Segmentation에 대한 깊이있는 이론 설명과 현업에서 바로 사용될 수 있는 많은 실습 예제들을 사전 학습한 후 스터디
* 11주간(22/04/16 - 22/06/22) 권철민 강사님의 '딥러닝 컴퓨터비전 완벽가이드' 강좌를 듣고 주 1회(토요일 오후 1:30 - 3:30) 발표
